# American Barber Institute — Website

Marketing & lead-generation website for **American Barber Institute (ABI)** — New York's only
dedicated barber school (est. 1996, NYS-licensed, Manhattan & Bronx campuses).

**Live:** https://abi-website-black.vercel.app
**Repo:** https://github.com/kazi-reprime/abi-website (default branch `main`)

---

## Tech & architecture

Zero-framework **static HTML**, generated by Python build scripts and hosted on Vercel.
There is **no runtime backend** — the Python runs only at build time.

| Script | Role |
|---|---|
| `build.py` | Merges `src/pages/*.html` partials into the base template → main pages + `sitemap.xml`/`robots.txt` |
| `build_landing_pages.py` | Builds the conversion landing pages (Manhattan / Bronx) + splash variants |
| `src/make_gallery.py` | Copies curated images into `assets/img` and generates the gallery partial |
| `src/make_blog.py` | Generates blog posts from `src/blog_manifest.json` |
| `src/deploy_vercel.py` | Uploads the static site to Vercel via REST (alt to CLI) |

- **Source of truth:** edit `src/pages/*.html`, then run `python3 build.py` — never edit the root `*.html` (they are regenerated).
- **CSS/JS** live in `assets/css` + `assets/js`, cache-busted via `?v=N` (bump on every CSS/JS change).
- **Per-page background photo** is set via `PAGE_BG` in `build.py` (each page gets its own hero image).

### Build & run
```bash
python3 build.py                 # build main pages
python3 build_landing_pages.py   # build landing pages
python3 -m http.server 8642      # preview at http://localhost:8642
```

### Deploy
```bash
vercel deploy --prod --yes       # (CLI, linked to project 'abi-website')
# or: VERCEL_TOKEN=... TEAM_ID=... python3 src/deploy_vercel.py
```

## Design system

- **Brand:** trust navy (`#0F172A` / `#142A4D`) + premium gold (`#F4C44A`) + barber red/blue accents.
- **Type:** Oswald (display) - Inter / Poppins (body).
- **Logo:** animated SVG — vertical ruler + continuously-rolling barber pole + "ABI" wordmark (reduced-motion safe).
- **Surfaces:** glassmorphic header & cards; clean white / subtle `.alt` content sections.
- **Backgrounds:** each page has its own content-matched hero photo (left-weighted gradient keeps text crisp); shared navy CTA band.
- **Motion:** glow + 3D micro-interactions, all gated by `prefers-reduced-motion`.
- **Accessibility:** WCAG-AA contrast, descriptive alt text, keyboard-accessible tour tabs, skip link.
